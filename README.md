<h2>Examples</h2>

* To list all the basic options and switches use -h switch:

```perl KnightSpy.pl -h```

* To Get Website Information:

``perl KnightSpy.pl -i example.com``

* To Get Phone Number Information :

``perl KnightSpy.pl -n xxxxxxx``

* To Find IP Address And E-mail Server:

``perl KnightSpy.pl -mx example.com``

* To Find Website Or IP Address Location :

``perl KnightSpy.pl -l example.com``

* To Get Real IP Of Website Using CloudFlare Protection

``perl KnightSpy.pl -c example.com``
